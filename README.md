

# 🎂 Dulce's Cakes — Landing Page

Landing page premium para **Dulce's Cakes**, pastelería artesanal venezolana.  
Desarrollada por [Quorex Studio](https://quorex.vercel.app).

🌐 **Live:** [dulcescakes.vercel.app](https://dulcescakes.vercel.app)

---

## Stack

- HTML5 + CSS3 puro (sin frameworks)
- Google Fonts: Playfair Display · Cormorant Garamond · DM Sans
- Animaciones con `@keyframes` CSS nativo
- Deploy en Vercel

## Secciones

- **Hero** — Logo animado, título, floating pills con productos, CTA a WhatsApp
- **Marquee** — Banda animada con todos los productos
- **Productos** — Grid de 13 postres con imagen real, nombre y descripción
- **Nosotras** — Historia de la marca y badges artesanales
- **CTA Final** — Botones directos a WhatsApp e Instagram
- **Footer** — Créditos y link a Instagram
